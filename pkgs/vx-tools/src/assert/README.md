# 类型判断函数

## 使用
```javascript

isString('xx')
// => true

```

## 方法
- [isString] 字符类型
- [isNumber] 数字类型
- [isBoolean] 布尔类型
- [isUndefined] undefined
- [isNull] null
- [isFunction] 函数类型
- [isObject] 对象类型
- [isArray] 数组类型

- [isEventName] 是否为事件名称 例如: 'onClick'

- [isType] 是否为某中类型 例如: isType('xx', 'string')
- [type] 获取数据类型 例如: type('xxx') // string

    对象过滤器

    summary

    通过关键字列表，抽取对象对应字段值，生成新的对象. 只做浅层映射

    example
    const source = {
  id: 12,
  name: 'coco',
  job: 'IT'
}
const keys = ['name', 'job']
const target = objectFilter(keys, source)
=> { name: 'coco', job: 'IT' }

Class Clock

计时器

summary

记录一段或多段时间间隔

example
 const c = new Clock // or Clock.create()
 
 c.tick()
 setTimeout(() => {
   c.tick()
 }, 1000)

 setTimeout(() => {
  console.log('time:', c.nextTick())
 }, 2000)
